Project Phases

Phase 1

Data collection is done using the Intel realsense camera for 2D data and Velodyne ultra-puck LiDAR camera for 3D data. After collecting data, we analyze it, clean, and label it for use in our web application.

Details

Phase 2

Machine learning models for both 2D and 3D data are trained with labeled data in order for the models to track and label vehicles in real-time. Data will stream through AWS clients to use in the web application.

Details

Phase 3

Website displays labeled real-time stream with graphs based on the data from 2D/3D Machine learning models. Registered users have access to historical data + camera configurations.

Details

Real-Time Traffic Monitoring Project

Our 491 team!

Our team has spent an entire year working on creating a platform that is able to provide real-time data streaming and statistics for traffic data. From machine learning model training, data collection, front/backend development, and data analysis, our team was able to create a way to add data sources like 2D/3D cameras and couple them with machine learning models to create a way to classify and analyze traffic efficiently.
